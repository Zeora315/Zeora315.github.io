Breek Pro文档

· 2026-02-21 ·

食用方法

  1. 下载主题文件下载主题文件下载主题文件
  2. 点击 Gridea 左侧「系统」菜单，查看源文件夹目录
  3. 将主题文件夹放入 Gridea 的 themes 目录
  4. 在 Gridea 客户端中选择「BreekPro」主题
  5. 根据需要进行自定义配置
  6. 点击「同步」部署到网站

支持的页面

  • 首页文章列表
  • 文章详情页
  • 归档页面
  • 标签页面
  • 搜索结果页
  • 即刻说说 - 类似微博的短内容页面
  • 书单 - 展示阅读过的书籍
  • 追番 - 展示追番列表
  • 友链 - 友情链接页面
